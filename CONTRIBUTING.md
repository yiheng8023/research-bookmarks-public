# Contributing

Contributions should improve public-safe taxonomy, official-source coverage, import/export contracts, validation, or documentation.

## Public-safe rule

Do not contribute private bookmarks, browsing history, account state, personal notes, private folders, subjective rankings, or non-official recommendations that have not been reviewed for public release.

## Source rule

Prefer official websites, official documentation, canonical source repositories, standards bodies, or clearly licensed public knowledge sources.

## Pull request checklist

- [ ] Sources are official or clearly marked for review.
- [ ] No personal browsing data is included.
- [ ] License and provenance are clear.
- [ ] `python -B scripts/verify.py` passes.
