# Private / Public Sync Model

Use public core plus private overlay.

```text
research-bookmarks-public
  -> taxonomy, official sources, import/export contracts

private bookmarks repository
  -> full bookmarks, private notes, non-official resources, preference data
```

Public-to-private sync is safe for taxonomy and official source seeds.

Private-to-public sync must be a filtered promote workflow. Do not mirror the private bookmark repository into the public repository.
